# SQL_Task_1

This project is a beginner to intermediate SQL practice exercise built around an orders database. It demonstrates core SQL concepts through structured questions and their corresponding solutions.

🔹 Dataset & Schema
- Created a database: practice
- Designed an orders table with fields like:
   -- order_id, cname, pname, category, qty, price_per_unit, order_date, delivery_status
- Inserted realistic sample data for multiple customers, products, and categories.

🔹 Key SQL Concepts Covered
- Aggregation Functions: SUM, AVG, MAX, COUNT
- Filtering & Conditions: WHERE, BETWEEN, IN, LIKE
- Logical Operations: AND, OR, NOT

- Data Manipulation (DML):
   --> UPDATE, DELETE, INSERT, SELECT
  
- Schema Modifications (DDL):
   --> ALTER TABLE (ADD, MODIFY, CHANGE columns)
  
- Derived Calculations:
   --> Revenue (qty * price_per_unit)

- Data Cleaning & Updates:
   --> Handling NULL values, default values
  
🔹 Example Queries
- Total revenue from delivered orders
- Orders within a date range
- Customers with specific name patterns
- Category-based price updates
- Adding new columns like payment_mode and total_amount

🔹 Files Included
- sql_Ex1_ques.sql → Contains schema + questions
- sql_Ex1_solutions.sql → Contains all query solutions
